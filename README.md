# 동산교회 청년부 웹페이지

## 파일 구조
```
├── index.html      # 웹페이지 본체 (건드릴 일 거의 없음)
├── content.json    # ✅ 콘텐츠 관리 파일 (여기만 수정하면 됨!)
└── README.md       # 이 파일
```

---

## ✅ 콘텐츠 수정 방법 (content.json)

### 모바일에서 수정하기 (GitHub 앱)
1. GitHub 앱 설치 → 이 저장소 접속
2. `content.json` 파일 터치
3. 연필 아이콘(✏️) 터치 → 내용 수정
4. 우측 상단 **Commit changes** 터치
5. 잠시 후 웹페이지 자동 반영 ✨

## 🚀 GitHub Pages 배포 방법 (최초 1회)

1. GitHub에서 새 저장소(Repository) 생성
2. 이 파일들 업로드
3. 저장소 → **Settings** → **Pages**
4. Source: **Deploy from a branch** → Branch: **main** → 저장
5. 잠시 후 `https://[계정명].github.io/[저장소명]/` 으로 접속 가능!
